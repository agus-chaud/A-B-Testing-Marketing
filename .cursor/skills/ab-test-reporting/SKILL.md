# ab-test-reporting

## Cuando usar

- Generar reportes de resultados de experimentos A/B.
- Crear visualizaciones (lift, intervalos de confianza, graficos de tendencia).
- Construir dashboards interactivos (Streamlit, etc.).
- Exportar a HTML, PDF o formatos compartibles.

## Skills relacionadas

- `ab-test-setup`: la hipotesis y las metricas (primaria, secundaria, guardrail) estructuran el reporte.
- `ab-test-stats`: incluir p-value, IC, tamano de muestra y conclusion estadistica.

## Checklist

1. Incluir metricas clave (lift, p-value, IC).
2. Visualizar distribuciones cuando aplique.
3. Respetar principios de accesibilidad (color-blind friendly).
4. Mantener formato consistente con reportes previos.
5. Incluir hipotesis cuando exista (Because/we believe); destacar metricas guardrail si aplican.
